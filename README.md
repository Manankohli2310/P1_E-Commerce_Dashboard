# 📊 E-Commerce Analytics Dashboard (P1)

An interactive **E-Commerce Analytics Dashboard** built using **Python, Pandas, Plotly, and Streamlit** for analyzing the `online_retail.csv` dataset from Kaggle.

This mini project focuses on exploring online retail sales data and visualizing important business insights such as revenue trends, top-selling products, customer activity, and order statistics.

---

## 🚀 Mini Project – Data Analysis

This project was created as a **Mini Project in Data Analysis** to practice:

- Data Cleaning
- Exploratory Data Analysis (EDA)
- Data Visualization
- Dashboard Development
- Business Insight Generation

---

## 📁 Dataset Used

Dataset: `online_retail.csv`  
Source: Kaggle Online Retail Dataset

The dataset contains transactional records of an online retail store including:

- Invoice details
- Product descriptions
- Customer IDs
- Quantity sold
- Unit prices
- Countries
- Dates of purchase
